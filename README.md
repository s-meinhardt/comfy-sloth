## Notes

#### Older React Version

```
"react": "^16.13.1",
"react-dom": "^16.13.1",
"react-scripts": "3.4.3",
```

#### Current React Version

```
"react": "^17.0.1",
"react-dom": "^17.0.1",
"react-scripts": "4.0.0",
```

### Environment Variables

You need to add an `.env` file with an Auth0 `REACT_APP_DOMAIN` and `REACT_APP_CLIENT_ID`.
